# 2048shooter - Technical & Architectural Specifications (V2)

이 문서는 `2048shooter` 게임의 아키텍처, 핵심 물리 수식, 상태 제어 시스템 및 밸런스 테이블을 명세합니다.

---

## 1. 개요 (Overview)
* **게임명**: 2048shooter
* **장르**: 2048 퍼즐 + 종스크롤 클래식 탄막 슈팅 (Bullet Hell)
* **엔진**: PixiJS v7.4.0 & Tailwind CSS (UI 레이어)
* **디자인 테마**: 기하학적 미니멀리즘 (Monochrome & Neon Geometry Outlines)

---

## 2. 핵심 물리 & 기계적 매커니즘 (Core Mechanics)

### 2.1. 등속 비행 및 조종 (Direct Control)
물리적 관성이나 지연으로 인한 조작 이질감을 완벽히 배제하기 위해 마우스 조작 시 1:1 즉시 매핑(Instant Follow)을 적용하고, 키보드 조작 시 가감속 없는 등속 비행을 채택합니다.

### 2.2. 기하학적 틸팅 회전 (Visual Tilting)
플레이어가 좌우로 기체를 이동할 때 역동감을 부여하기 위해 수평 비행 속도($\Delta x$)에 비례하여 기체 그래픽이 좌우로 부드럽게 기울어집니다.
* $dx > 0$ 이면 오른쪽으로 기울어짐 (Max 0.35 rad)
* $dx < 0$ 이면 왼쪽으로 기울어짐 (Min -0.35 rad)
